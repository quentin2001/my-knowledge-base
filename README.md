# my-knowledge-base

# 📖 纯粹笔记 (Pure Note)

> **将 Notion 的极简美学，与 Obsidian 的本地掌控感完美融合。**

## 💡 开发契机与产品哲学 (Why we built this)

在如今的生产力工具市场，我们常常面临两难的选择：

  * **Notion** 拥有令人叹为观止的排版美学和极简 UI，但它高度依赖云端，加载缓慢，且日益臃肿（被强制塞入了许多我们不需要的多人协作和 AI 功能）。
  * **Obsidian** 给予了我们绝对的本地文件掌控权，但它的默认 UI 过于硬核，且折腾美化插件往往会耗费大量精力。

**我们需要一个“刚刚好”的软件。**

本项目诞生的初衷，就是为个人知识工作者打造一款**拒绝功能焦虑**的桌面笔记软件：

1.  **极致的 UI 审美**：复刻 Notion 级别的高级质感、排版与深浅色模式。
2.  **绝对的数据掌控**：像 Obsidian 一样基于本地文件树（Markdown）管理，你的数据永远属于你。
3.  **功能做减法**：无广告、无订阅、无强制联网、无冗余的多人协作与 AI 模块。你需要的核心功能做到极致，你不需要的负担通通砍掉。

-----

## ✨ 核心特性 (Features)

目前项目已完成核心框架与高频使用场景的建设：

  - 🌗 **原生级双端主题**：内置精心调配的 Notion 风格色彩系统，支持一键无缝切换深色/浅色模式。
  - 📝 **沉浸式块级书写**：基于 Tiptap 打造的富文本 Markdown 编辑器，支持流畅的所见即所得体验、双列布局及动态大纲（TOC）联动跳转。
  - 🗂️ **左侧文件管理树**：完全映射本地操作系统的文件夹结构。支持多级目录嵌套、拖拽排序、右键菜单重命名与删除。
  - 🖼️ **优雅的资产管理**：支持图片的直接粘贴与拖拽存入，自动在本地归档并渲染，告别图床焦虑。
  - 🪟 **多窗口与导入导出**：支持同时打开多个独立的本地文件夹工作区；支持单页面导出或整个知识库文件夹的无损迁移。

-----

## 🗺️ 路线图 (Roadmap)

我们正在稳步推进将其打造为“商业级应用”的最后几步，接下来的开发重点如下：

  - [ ] **无感自动保存 (Auto-save)**：引入防抖（Debounce）机制，实现用户停笔 1 秒级后台静默保存，保障数据绝对安全。
  - [ ] **全局秒级搜索 (Global Search)**：基于本地文件系统的全知识库文本检索，让你瞬间定位灵感碎片。
  - [ ] **UI 细节打磨 (UI/UX Polish)**：持续优化各类交互动画（如侧边栏收缩、弹窗挤压）、字体排印（Typography）以及留白比例，进一步提升使用愉悦感。

-----

## 🛠️ 技术栈 (Tech Stack)

  * **底层框架**：Electron (支持全平台打包)
  * **前端框架**：Vue 3 (Composition API) + Vite
  * **核心编辑器**：Tiptap (基于 ProseMirror)
  * **开发语言**：TypeScript 严格模式
  * **文件系统**：Node.js `fs-extra` + 自定义 Preload 安全桥梁
